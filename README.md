# lumo-practice
A project of lumo for my practice about clojurescript.

# Setup
## lumo
Install [lumo](https://github.com/anmonteiro/lumo).

## dependencies
```
npm install
```
# Run
## Hello world
```
lumo -c src -m hello-world.core
```

## Express sample app
```
lumo -c src -m express-sample.core
```

Then access to http://localhost:3000
